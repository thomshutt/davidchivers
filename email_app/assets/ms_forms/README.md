# Microsoft Forms visual assets

These assets are the final simplified Durham-branded visuals for the student enquiry form.

Files:

- `durham_forms_header.png`
  Use as the header image in the form title area.
- `durham_forms_background.png`
  A plain white background image if you want to upload one.
- `durham_forms_background_purple.png`
  Solid Durham purple background (`#68246D`) for use as a simple Forms background.
- `durham_forms_background_68246D.png`
  Larger solid Durham purple background (`#68246D`) if Forms prefers a bigger image.
- `durham_forms_background_logo_bottom_right.png`
  Recommended watermark-style background. Durham mark sits low/right so it can be cropped by Microsoft Forms without looking broken.
- `durham_forms_background_logo_top_right.png`
  Alternative watermark-style background with the Durham mark higher/right.
- `durham_forms_background_logo_panel.png`
  More literal logo-panel background. Use only if Microsoft Forms crops the top-left area nicely in testing.
- `durham_logo.png`
  Source logo image used to build the header.

Suggested Forms settings:

- Header image: `durham_forms_header.png`
- Accent color: `#68246D`
- Text color: black
- Background: white
- Background image: optional, `durham_forms_background.png`
- Alternative background image: `durham_forms_background_logo_bottom_right.png`

Notes:

- The final header is intentionally simple:
  white background, Durham logo, and `Department of Economics` in black.
- There are no yellow accents or decorative shapes in the final version.
- If you want the cleanest Microsoft Forms look, upload the header image, set the accent color to Durham purple, and leave the rest of the form white.
- If Microsoft Forms behaves better with a background than a header, use one of the solid purple background files and keep the rest of the styling minimal.
- Avoid relying on pixel-perfect logo placement in the Microsoft Forms background. Forms may crop and scale the image differently on desktop and mobile. A subtle watermark placed low/right is safer than a precise visible logo.
- Suggested form title: `Student enquiry form`
- Suggested description: `Use this form to contact the Department of Economics about your enquiry.`
