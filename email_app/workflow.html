Email App Instructions

This page explains the current live pilot, what is already working, and what still changes during testing.

Current live setup

  • The Microsoft Form is already built.
  • The Excel lookup workbook is already built.
  • The Power Automate flow is already built.
  • Right now, staff notification emails go to david.chivers@durham.ac.uk.
  • That is the pilot inbox for testing, not the final destination.
  • After testing, the same flow can be moved to another inbox or to a shared mailbox.

If you are testing the current setup, start with Use email app. If you want to change where the staff email lands, start with Update email app.
